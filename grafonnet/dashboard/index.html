    Choose how to sort the values in the dropdown.

    This can be called as withSort(<number>) to use the integer values for each option. Ifi==0` then it will be ignored and the other arguments will take precedence.

    The numerical values are:

    • 1 - Alphabetical (asc)
    • 2 - Alphabetical (desc)
    • 3 - Numerical (asc)
    • 4 - Numerical (desc)
    • 5 - Alphabetical (case-insensitive, asc)
